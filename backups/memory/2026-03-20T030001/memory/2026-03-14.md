## Lessons
- MC memory graph ≠ QMD index — different schemas, different locations. MC reads from `~/.openclaw/memory/*.sqlite` (old) or QMD indexes (new).
- OpenClaw config has **two levels** of group allowFrom: top-level `groupAllowFrom` (what doctor checks) and nested `groups["*"].allowFrom` (per-group overrides)
- **NEVER guess openclaw.json field names/structure.** Always check docs (`docs.openclaw.ai`), doctor output, or existing config. John's explicit hard rule after gateway crashes from guessed edits.
