# GitHub Crypto Trading Bot Research — 2026-03-29

## Top Repos Analyzed

### 1. freqtrade/freqtrade ⭐ 48,164

## Biggest Gaps in V2

1. ❌ Volatility-based position sizing — no ATR/σ in decision loop
2. ❌ Inventory management — no skewing based on current position
3. ❌ Market regime detection — doesn't know trending vs ranging
4. ❌ Time-of-day awareness — no session filtering
5. ❌ Backtesting engine — can't validate on historical data
6. ❌ Performance metrics — no Sharpe, Sortino, max DD tracking
7. ❌ Adaptive model — static rules, no self-learning
