### Session 4: Service restart + hotfixes
- ai-trader crashed on restart: `SyntaxError: 'await' outside async function` in `_emergency_halt`
  - Fix: made `_emergency_halt` async, added `await` at both call sites (commit `8674014`)
- `ModuleNotFoundError: No module named 'ipc_utils'` — path was `.parent.parent / "shared"` but should be `.parent.parent.parent`
  - Fix: corrected in both `ai_trader.py` and `context_builder.py` (commit `bd5d4be`)
- `AttributeError: 'SafetyLayer' has no attribute '_last_failure_time'` — introduced by subagent in commit `51397e6` (kill switch decay feature)
  - Subagent added code that READS `_last_failure_time` but never INITIALIZED it
  - Also found `failure_decay_minutes` and `rejection_halt_window_minutes` used but never initialized
  - Fix: added all 3 missing attributes to `__init__` (commit `510421f`)
- Added `__slots__` to SafetyLayer — prevents future undeclared attribute bugs (commit `82ec633`)
- Added `ExecStartPre` healthcheck to ai-trader.service — import check before start
- gitignored `.ack` runtime files (commit `0583f3c`)

**Lessons:**
- Subagents can introduce subtle bugs (missing __init__ attributes) that only surface on restart
- `__slots__` catches this class of bug at write time
- ExecStartPre healthcheck prevents cascading restart loops
- Always restart services after merging fixes to catch these
