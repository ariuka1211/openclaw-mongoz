## Summary

Tool that turns documentation websites into a browsable filesystem for coding agents. Like browsing a code repo instead of web-searching.

> "Code hallucinations are not a model problem. They're a data problem."

Claims 41% improvement in coding agent accuracy with latest frameworks (Next.js).

## Connection to LLM Wiki

Same principle as Karpathy's pattern but applied to external knowledge (live docs) rather than personal knowledge. A wiki that's always up-to-date with current docs, not stale training data.
