# Arlan's AgentSearch

**Original:** [Tweet by Arlan](https://x.com/arlanr/status/2041233249180225622), 2026-04-06

## Summary

Tool that turns documentation websites into a browsable filesystem for coding agents. Claims 41% improvement in coding agent accuracy with latest frameworks.

## Key Quote

> "Code hallucinations are not a model problem. They're a data problem."

## Connection to LLM Wiki Pattern

Same principle as Karpathy's approach but applied to external knowledge (live docs) rather than personal knowledge. Instead of models relying on stale training data, give them structured access to current documentation.
