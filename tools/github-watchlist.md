### Lightpanda Browser
- **What:** Headless browser in Zig. 9x less memory, 11x faster than Chrome. CDP compatible (Playwright/Puppeteer drop-in).
- **Why:** Heavy scraping/automation at scale. When web_fetch isn't enough and Chromium is too heavy.
- **Status:** Nightly only, partial Web API support (WIP)
- **Link:** https://github.com/lightpanda-io/browser
- **Found:** 2026-03-14
